How to sign a PDF

Select the file you want to sign. Create a signature with the given possibilities. Place the signature. Create and save the signed PDF.

Create a signature

You can create a signature in different ways. Everything is included, from creation via mouse or touchpad, upload as image or creation via camera.

Ease of use

We make it as easy and fast as possible to sign files electronically. You don't have to install or set anything, just sign your file here.

Supports your system

There are no special requirements for signing files on your system. This app works with all common operating systems and browsers.

No installation required

You do not need to install any software. This app runs on our servers in the cloud and your system does not require any special requirements.

Security is important to us

This signature tool does not store your files on our server for longer than necessary. Your files will be removed from our system after a short time.
